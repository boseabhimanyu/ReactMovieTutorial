## TMDB API Integration with React


Fetching popular movie details from TMDB API. The webpage has search movies option and favourite marking movies feature.


# Setup

Clone the repo

```
git clone https://github.com/boseabhimanyu/ReactMovieTutorial.git
```

```
cd frontend
```


Create a new .env file

```
npm install
```

```
npm run dev
```

# `.env` file variables

```
VITE_API_KEY=<API KEY>
```

```
VITE_API_URL=https://api.themoviedb.org/3
```
